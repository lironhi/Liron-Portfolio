# COVID-19 Information & Appointment System

A comprehensive web platform developed during the COVID-19 pandemic to provide accessible information and streamline the testing appointment process. Built with Django, this system combines a public-facing informational website with a robust CRM for healthcare management.

## üéØ Project Context

**Challenge**: During the pandemic, people needed easy access to COVID-19 information and testing appointments, while healthcare facilities struggled to manage high volumes of bookings.

**Solution**: An integrated platform providing:
- Up-to-date COVID-19 information
- Testing center locator with real-time availability
- Online appointment booking
- CRM system for healthcare workers
- Patient management and communication tools

# views.py - Find nearest testing centers
def find_centers(request):
    user_location = request.GET.get('location')
    radius = request.GET.get('radius', 10)  # km

    centers = TestingCenter.objects.filter(
        is_active=True,
        available_slots__gt=0
    ).annotate(
        distance=Distance('location', user_location)
    ).filter(
        distance__lte=radius
    ).order_by('distance')

    return render(request, 'centers/list.html', {
        'centers': centers
    })
```

    appointment_date = models.DateTimeField()
    status = models.CharField(max_length=20, choices=STATUS_CHOICES)
    test_result = models.CharField(max_length=20, null=True, blank=True)
    notes = models.TextField(blank=True)

    class Meta:
        ordering = ['appointment_date']

class Patient(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    phone_number = models.CharField(max_length=20)
    date_of_birth = models.DateField()
    medical_conditions = models.TextField(blank=True)
    vaccination_status = models.CharField(max_length=50)

# utils/notifications.py
from twilio.rest import Client
from django.core.mail import send_mail

def send_appointment_confirmation(appointment):
    # SMS notification
    client = Client(settings.TWILIO_SID, settings.TWILIO_TOKEN)
    message = client.messages.create(
        body=f"Your COVID-19 test is scheduled for {appointment.appointment_date}",
        from_=settings.TWILIO_NUMBER,
        to=appointment.patient.phone_number
    )

    # Email confirmation
    send_mail(
        subject='COVID-19 Test Appointment Confirmation',
        message=render_to_string('emails/confirmation.html', {
            'appointment': appointment
        }),
        from_email=settings.DEFAULT_FROM_EMAIL,
        recipient_list=[appointment.patient.user.email]
    )
```
