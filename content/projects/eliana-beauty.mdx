# Eliana Beauty - Booking & CRM System

A comprehensive, mobile-first booking and customer relationship management platform designed specifically for beauty salons and independent beauty professionals. The system integrates client-facing booking capabilities with powerful CRM tools for managing appointments, staff schedules, services, and business analytics.

## Architecture & Technology Stack

The project employs a modern fullstack TypeScript monorepo architecture using **pnpm and Turborepo** for efficient workspace management.

### Backend Infrastructure

- **NestJS 10** - Enterprise-grade API development framework
- **Prisma 5** - Type-safe database ORM
- **PostgreSQL** - Relational database
- **JWT Authentication** - Refresh token rotation
- **bcryptjs** - Password hashing (10 rounds)
- **class-validator** - Request validation

### Frontend Stack

- **React 18** - Modern hooks and functional components
- **Vite 5** - Rapid development and optimized builds
- **TypeScript 5** - Full type safety
- **Tailwind CSS 3** - Utility-first styling
- **Zustand** - Lightweight state management
- **React Router 6** - Client-side navigation
- **Vite PWA** - Progressive web app capabilities
- **react-hot-toast** - Elegant notifications
- **Recharts** - Data visualization

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- pnpm >= 8.0.0
- Docker and Docker Compose for PostgreSQL

### Installation Steps

1. Clone the repository
2. Install dependencies via `pnpm install`
3. Create environment files from provided examples
4. Start PostgreSQL with `pnpm db:up`
5. Configure database with schema push and seeding
6. Run development servers with `pnpm dev`

### Default Credentials

- **Admin:** admin@eliana.beauty / admin123
- **Staff:** eliana@eliana.beauty / staff123
- **Client:** test@example.com / test123
