# Smart Kitchen Mobile App

A mobile application designed to help users manage their kitchen inventory efficiently through automated receipt scanning, ingredient tracking, and intelligent recipe suggestions.

## ðŸŽ¯ Problem & Solution

**Problem**: Managing kitchen ingredients is tedious - tracking what you have, when it expires, and what recipes you can make.

**Solution**: An Android app that automates ingredient tracking through receipt OCR and provides intelligent recipe recommendations.

## ðŸ”‘ Key Features

### 1. OCR Receipt Scanning
- **Scan receipts** using device camera
- **Extract ingredients** automatically using Google ML Kit
- **Parse quantities** and prices from receipt text
- **Add to inventory** with one tap

```java
// OCR Implementation with ML Kit
TextRecognizer recognizer = TextRecognition.getClient(TextRecognizerOptions.DEFAULT_OPTIONS);

Task<Text> result = recognizer.process(image)
    .addOnSuccessListener(text -> {
        parseIngredients(text.getText());
    })
    .addOnFailureListener(e -> {
        handleError(e);
    });
```

### 2. Ingredient Management
- Track quantities and units
- Monitor expiration dates
- Automated low-stock alerts
- Barcode scanning for quick entry

### 3. Recipe Suggestions
- Suggests recipes based on available ingredients
- Filters by missing ingredients count
- Saves favorite recipes
- Shopping list generation for missing items

### 4. Firebase Integration
```java
// Firebase Realtime Database
FirebaseDatabase database = FirebaseDatabase.getInstance();
DatabaseReference ref = database.getReference("users/{userId}/ingredients");

// Add ingredient
ref.child(ingredientId).setValue(new Ingredient(
    name, quantity, unit, expirationDate
));

// Real-time updates
ref.addValueEventListener(new ValueEventListener() {
    @Override
    public void onDataChange(DataSnapshot snapshot) {
        updateUI(snapshot.getValue());
    }
});
```
