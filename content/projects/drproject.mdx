# DrProject - Blood Test Analysis Platform

A web application designed to help patients understand their blood test results and identify potential health concerns based on medical data analysis.

## üéØ Problem & Solution

**Problem**: Blood test results are often confusing for patients, with complex medical terminology and reference ranges that are hard to interpret.

**Solution**: An intuitive web platform that analyzes blood test parameters, explains what they mean, and provides insights into potential health issues.

## üîë Key Features

### 1. Blood Test Analysis
- **Upload test results** manually or via form input
- **Compare with normal ranges** for each parameter
- **Visual indicators** (color-coded) for abnormal values
- **Detailed explanations** of each blood marker

### 2. Health Insights
- Identify potential health issues based on abnormal values
- Provide educational content about each parameter
- Suggest follow-up tests or medical consultation when needed
- Track changes over time

### 3. User-Friendly Interface
- Clean, modern design with CSS3
- Responsive layout for mobile and desktop
- Interactive charts and visualizations
- Easy navigation between different test categories

## üìä Technical Implementation

### Frontend Stack
- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with flexbox/grid
- **JavaScript**: Interactive features and data visualization
- **Vanilla JS**: No framework dependencies for lightweight performance

### Core Functionality
```javascript
// Blood test parameter analysis
function analyzeParameter(value, normalRange) {
  const { min, max } = normalRange;

  if (value < min) {
    return {
      status: 'low',
      severity: calculateSeverity(value, min),
      recommendations: getLowRecommendations(parameter)
    };
  } else if (value > max) {
    return {
      status: 'high',
      severity: calculateSeverity(value, max),
      recommendations: getHighRecommendations(parameter)
    };
  }

  return { status: 'normal' };
}
```

- Platelets

### Metabolic Panel
- Glucose
- Creatinine
- Blood Urea Nitrogen (BUN)
- Electrolytes (Na, K, Cl)

### Lipid Panel
- Total Cholesterol
- LDL (Bad cholesterol)
- HDL (Good cholesterol)
- Triglycerides

### Liver Function
- ALT (Alanine Aminotransferase)
- AST (Aspartate Aminotransferase)
- Bilirubin
- Alkaline Phosphatase
